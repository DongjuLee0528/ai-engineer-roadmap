# 조건부 확률 (Conditional Probability)

> Fast Campus | 선수 지식 - 통계 | 강사: 나동빈

---

## 1. 개념 정의

- **어떠한 사건이 일어나는 경우에 다른 사건이 일어날 확률**을 의미한다.
- 딥러닝 분야에서는 **"X 사건이 단서로 주어졌을 때, Y 사건이 발생할 확률"** 로 이해할 수 있다.

---

## 2. 딥러닝에서의 활용 — 분류 모델

분류 모델은 입력 이미지 $x$가 주어졌을 때, 각 클래스 $y$가 나올 확률을 조건부 확률로 표현한다.

$$P(Y = y \mid X = x)$$

### 예시

| 클래스 | 확률 |
|--------|------|
| $P(Y = \text{코끼리} \mid X = x)$ | 0.87 |
| $P(Y = \text{새} \mid X = x)$ | 0.06 |
| $P(Y = \text{벌} \mid X = x)$ | 0.07 |

---

## 3. 확률의 종류

### 예시 데이터

|  | 스팸 메일 | 일반 메일 | 합계 |
|--|----------|----------|------|
| 학교 계정 | 40 | 30 | 70 |
| 회사 계정 | 50 | 60 | 110 |
| **합계** | **90** | **90** | **180** |

### 주변 확률 (Marginal Probability)

하나의 메일을 뽑았을 때, 학교 계정으로 온 메일일 확률

$$P(\text{학교}) = \frac{70}{180}$$

### 결합 확률 (Joint Probability)

하나의 메일을 뽑았을 때, 학교 계정이면서 동시에 스팸 메일일 확률

$$P(\text{학교} \cap \text{스팸}) = \frac{40}{180}$$

### 조건부 확률 (Conditional Probability)

스팸 메일 중 하나를 뽑았을 때, 학교 계정으로 온 메일일 확률

$$P(\text{학교} \mid \text{스팸}) = \frac{40}{90}$$
