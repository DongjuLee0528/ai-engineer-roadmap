# 📊 연속확률분포 (Continuous Probability Distribution)

> 딥러닝의 기초가 되는 확률 개념 | Fast Campus - 나동빈 강사

---

## 1️⃣ 연속확률분포란?

- **연속확률변수**: 확률변수 $X$가 취할 수 있는 값이 **무한한** 경우
- **연속확률분포**: 연속확률변수의 확률 분포
- 셀 수 없이 많은 확률 변수들의 분포이므로, **특정 값 $x$에 대한 정확한 확률 값을 표현할 수 없다**
- 따라서 특정한 **구간** $a \leq x \leq b$ 에 대한 확률로 표현

---

## 2️⃣ 확률밀도함수 (PDF, Probability Density Function)

- 연속확률변수가 **주어진 구간 내에 포함될 확률**을 출력하는 함수
- 참고: 이산확률변수는 PDF가 아닌 **확률질량함수(PMF)** 를 사용

### 📌 연속확률분포의 성질

| 성질 | 수식 | 설명 |
|------|------|------|
| 전체 면적 = 1 | $\int_{-\infty}^{\infty} f(x)dx = 1$ | 모든 구간의 확률 합은 1 |
| 구간 확률 | $P(a \leq X \leq b) = \int_a^b f(x)dx$ | a부터 b까지의 면적 |

### 💡 확률밀도함수 해석
- $f(x)$는 **확률**, $dx$는 **구간 길이**를 의미
- "확률 / 구간 길이"의 값을 모두 더한 것 → **밀도(density)** 개념
- 구간 길이를 부피로 볼 때, "질량 / 부피" = 밀도

### ⚠️ 참고 사항
- 연속 확률 변수는 면적으로 계산
- **한 점에 대한 확률은 0**으로 간주
- 따라서: $P(a \leq X \leq b) = P(a < X < b)$

---

## 3️⃣ 균등 분포 (Uniform Distribution)

- 특정 구간 내 값들이 나타날 가능성이 **균등**한 분포
- 모든 확률변수에 대해 **일정한 확률**을 가짐
- 표기: $X \sim U(a, b)$
- $P(a \leq X \leq b) = 1$

### 📐 확률밀도함수

$$f(x) = \begin{cases} \dfrac{1}{b-a}, & a \leq X \leq b \\ 0, & \text{otherwise} \end{cases}$$

### 📝 예시 1: 스팸 메일
- 매일 오후 3시~4시 사이에 스팸 메일이 균등한 확률로 도착
- 단위를 "분"으로 하면 $f(x) = \dfrac{1}{60}$ (직사각형 모양)

### 📝 예시 2: 계산 문제
> **문제**: $X$가 구간 $[10, 20]$에서 균등 분포일 때, $P(15 \leq X \leq 17)$은?

$$f(x) = \frac{1}{20-10} = \frac{1}{10}$$

$$P(15 \leq X \leq 17) = \frac{1}{10} \times 2 = \frac{1}{5}$$

### 📌 이산 균등 분포 (Discrete Uniform Distribution)
- 균등 분포는 이산확률변수에도 정의 가능
- 예: 주사위 1회 시행 → $P(X=k) = \dfrac{1}{6}$ for $k = 1, 2, 3, 4, 5, 6$

## 4️⃣ 정규 분포 (Normal Distribution)

- 기계학습에서 **매우 자주** 등장하는 분포
- **가우시안(Gaussian) 분포**라고도 불림
- 분포 모양: 평균을 중심으로 **좌우 대칭인 종(bell) 모양**

### 📐 확률밀도함수

$$f(x) = \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{1}{2\sigma^2}(x-\mu)^2}$$

- $\mu$: 평균 (분포의 위치 결정)
- $\sigma^2$: 분산 (분포의 퍼짐 결정)
- 범위: $-\infty < x < \infty$, $\sigma > 0$

### 📌 정규 분포의 특징

| 파라미터 | 효과 |
|----------|------|
| **평균 $\mu$ 변화** | 분포가 좌우로 **평행이동** |
| **분산 $\sigma^2$ 클수록** | 분포가 옆으로 **넓게 퍼짐** |
| **분산 $\sigma^2$ 작을수록** | 분포가 **가파른** 모양 |

- 관측 값의 약 **98%**가 $\pm 2\sigma$ 범위 안에 속함

### 📌 표준 정규 분포

- 평균 = 0, 표준편차 = 1인 정규 분포
- 정규 분포 함수를 직접 적분하기 매우 어려움
- 실제로는 **표준 정규 분포로 변환** 후 확률 계산

### 💡 왜 정규 분포가 중요한가?
- 현실 세계의 많은 데이터가 정규 분포로 **가깝게 표현**되는 경향
- 수집된 데이터의 **분포를 근사**할 때 자주 사용
- 생성 모델: 이미지의 분포를 정규 분포로 근사 → 있을 법한 이미지 생성 가능

## 5️⃣ 지수 분포 (Exponential Distribution)

- 특정 시점에서 어떤 사건이 **일어날 때까지 걸리는 시간**을 측정할 때 사용
- 포아송 분포와의 관계:

| 분포 | 종류 | 목적 |
|------|------|------|
| **포아송 분포** | 이산 확률 분포 | 발생 **횟수**에 대한 확률 |
| **지수 분포** | 연속 확률 분포 | **대기 시간**에 대한 확률 |

### 📐 확률밀도함수

$$f(x) = \lambda e^{-\lambda x}, \quad x \geq 0$$

- $\lambda$: 단위 시간 동안 **평균 사건 발생 횟수**

### 📝 예시: 해킹 시도
> **문제**: 서버에 하루 평균 4건의 해킹 시도 → 첫 번째 해킹이 3시간 안에 발생할 확률은?

**풀이:**
- 24시간에 4건 → $\lambda = \dfrac{4}{24}$
- 확률 밀도 함수: $f(t) = \dfrac{4}{24} e^{-\frac{4}{24}t}$

$$\int_0^3 \frac{4}{24} \exp\left(-\frac{4}{24}t\right)dt = \left[-\exp\left(-\frac{4}{24}t\right)\right]_0^3$$

$$= \exp(0) - \exp\left(-\frac{4}{24} \times 3\right) \approx 0.3935$$

### ⚠️ 지수 분포의 특성: 무기억성 (Memoryless)
- 특정 시점에서부터 소요되는 시간은 **과거로부터 영향을 받지 않음**
- 예: 서버를 3년 운영한 뒤 해킹까지 걸리는 시간 = 처음 운영 시 해킹까지 걸리는 시간
- **한계**: 현실 세계의 다양한 사례를 모델링하기에는 지나치게 단순

| 분포 | 종류 | 특징 | 용도 |
|------|------|------|------|
| **균등 분포** | 연속 | 모든 값이 동일한 확률 | 특정 구간 내 균등 발생 사건 |
| **정규 분포** | 연속 | 종 모양, 평균·분산으로 결정 | 현실 데이터 근사, ML 전반 |
| **지수 분포** | 연속 | 무기억성, 감소 형태 | 대기 시간, 수명 분석 |
| **이산 균등 분포** | 이산 | 모든 값이 동일한 확률 | 주사위, 카드 등 |
